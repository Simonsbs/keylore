# Code of Conduct

## Our standard

Participants in this project are expected to act professionally, stay technical, and keep collaboration respectful.

## Expected behavior

- discuss ideas and code directly
- provide evidence for technical claims
- accept review feedback without hostility
- keep security-sensitive discussion responsible and discreet

## Unacceptable behavior

- harassment, insults, or intimidation
- publishing private information without consent
- intentionally disruptive conduct
- deliberately introducing malicious code or secret leakage

## Enforcement

Project maintainers may remove comments, reject contributions, or block participation when behavior materially undermines a safe and effective engineering environment.
