Local-first install

Default setup takes two commands.

npm install -g @simonsbs/keylore
keylore-http start

Then open http://127.0.0.1:8787/. KeyLore runs in the background and stores writable state under ~/.keylore.

What happens next

First-run checklist.

1

Add a token

Store the real secret in KeyLore instead of leaving it in shell env or project files.

2

Add context

Write human context and LLM context so the agent can pick the credential by purpose.

3

Test and connect

Run the brokered token test, then connect Codex, Gemini CLI, Claude CLI, or another MCP client.
