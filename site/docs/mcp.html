Selection model

Agents should choose by context fit, not token name similarity.

Search metadata first

Use service, domain, permitted operation, tags, and context to find the best candidate.

Inspect the record

Read the metadata record to confirm it matches the intended task and that the domains and operations are appropriate.

Request brokered access

Use the live access path only after the credential has been selected by context, not by guesswork.
