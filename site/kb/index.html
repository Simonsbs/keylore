Top questions

Common answers for setup and migration.

How does KeyLore stop the agent from seeing the raw token?

The agent searches metadata and requests access through MCP. KeyLore keeps the secret in a separate store and brokers the live action without returning the raw token value.

Can I start local-first and harden later?

Yes. The default install is designed for fast local adoption. Later you can move to stronger deployment controls, external secret stores, and wider team governance.

How do I migrate from .env files?

Install and start KeyLore, move the token into KeyLore, add minimal metadata, connect the MCP client, verify brokered access works, then remove and rotate the old environment-based secret so the shell path is no longer trusted.

What should the agent search for?

Service, domain, permitted operation, tags, and usage context. The agent should prefer the best context match, not just the most familiar token name.
