Migration path

Move the token, then remove the old trust path.

1

Install and start KeyLore

Run the local-first install and open the UI.

2

Add the token

Import the existing token into KeyLore and give it a unique token key.

3

Add context

Write the human and LLM context that tells the agent when the token is the right choice.

4

Rotate away from env

After the MCP flow works, remove the old environment-based secret and rotate it so the old path is no longer trusted.
