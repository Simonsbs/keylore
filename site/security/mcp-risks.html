Threat surface

Agent tooling is powerful. That makes its credential model important.

Prompt injection

An agent can be persuaded to use the wrong tool or overreach unless credentials are described and constrained well.

Confused deputy behavior

A powerful token plus a weak selection model can let the agent do more than the task actually requires.

Overshared secret material

If the token is directly available to the process, any failure mode can turn into a leak path.

Design response

What KeyLore changes.

Metadata-first selection

Agents choose credentials by service, domain, operation, and context instead of seeing raw secrets.

Brokered execution

The live action goes through KeyLore, which keeps the token value out of the prompt path.
