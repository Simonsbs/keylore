Why this setup matters

Codex should ask KeyLore for access, not read your token directly.

Point Codex at KeyLore instead of handing Codex a raw token. Codex searches metadata, requests brokered access, and never receives the secret value. 

# ~/.codex/config.toml
[mcp_servers.keylore]
command = "keylore-stdio"
